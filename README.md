# Vibe Tasks ✦

Vibe Tasks is a beautifully designed, front-end To-Do list application that aims to make your daily productivity colorful, animated, and joyful!

## 🌟 Features

- **Aesthetic UI**: A vibrant rainbow-gradient glassmorphism design with an interactive bubbly aesthetic.
- **Dark/Light Themes**: Easily toggle between fully styled Dark and Light viewing modes.
- **Task Management**:
  - Add tasks easily with an interactive input interface.
  - Tag tasks with four priority levels (🔥 Urgent, ⚡ High, 💧 Normal, 🌿 Low).
  - Inline editing capability for quick corrections.
  - Quick-toggle checkboxes that trigger celebration effects.
  - Delete or clear out completed tasks.
- **Progress Tracking**: See a calculated percentage and progress bar for completed tasks vs pending ones.
- **Filter Views**: Switch views to see "All", "Pending", or "Done" tasks with dynamic counts.
- **Micro-interactions**: Enjoy bouncy transitions, custom floating blobs in the background, tooltips, and custom toast notifications.
- **Celebration Confetti**: Because finishing a task should be celebrated! 🎉
- **Local Storage Functionality**: Your tasks are saved securely in your browser's local storage so they're waiting for you the next time you open the app.
- **Responsive Design**: Built to look great and function seamlessly on both desktop and mobile views.

## 🛠️ Built With

This project relies on pure vanilla web technologies with no heavy frameworks required:
- **HTML5** (Structure)
- **CSS3** (Styling, Keyframe Animations, Glassmorphism Filters, CSS Variables)
- **Vanilla JavaScript / ES6** (DOM Manipulation, State Management, LocalStorage)

### External Assets used:
- **Fonts**: [Nunito](https://fonts.google.com/specimen/Nunito) and [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) from Google Fonts.
- **Icons**: [Font Awesome 6.5.0](https://fontawesome.com/)

## 🚀 How to Run

Because this is a static web application built perfectly into a single file, it's incredibly simple to use!

1. Clone this repository or download the `index.html` file.
2. Open the `index.html` file in any modern web browser.
3. Start adding tasks and enjoying the productivity vibes!
